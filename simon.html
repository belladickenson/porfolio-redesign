Simon Game

This project showcases my skills in front-end web development, particularly in HTML, CSS, and JavaScript. I designed and developed a fully functional Simon game, a classic memory game that challenges players to repeat a sequence of sounds and colors in increasing difficulty.

To achieve this, I utilized my expertise in HTML and CSS to craft an eye-catching and intuitive user interface that closely mimics the look and feel of the original Simon game. I then implemented JavaScript to create the logic for the game, generating random sequences of colors and sounds, tracking player progress, and providing feedback and scoring.

Throughout the development process, I paid close attention to details such as animation and sound effects to create an engaging and immersive user experience. The end result is a fully functional and visually appealing Simon game that effectively showcases my proficiency in front-end web development.
