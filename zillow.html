Zillow Clone

I created a static design replica of the Zillow homepage and search page to showcase my skills in HTML and CSS. Although the website is not functional, it accurately captures the look and feel of the original Zillow website.

Visitors to the site can explore the static maps, property listings, and search filters, gaining a sense of the website's functionality as they navigate. While the website lacks back-end functionality, its attention to detail and user experience demonstrate my front-end development skills and design capabilities.

Creating this Zillow clone allowed me to hone my skills in front-end web development, including HTML5, CSS3, and responsive design techniques. This project also gave me the opportunity to refine my ability to replicate complex interfaces using only static files.
