React Notes App

As a front-end developer, I designed and developed a note taking application using Javascript and React. This application allows users to create and organize notes or reminders.

With a simple and intuitive user interface, users can quickly add and edit notes and view their notes in a list or grid format. By building this note taking application, I gained experience in working with React components and state management. I also honed my skills in implementing responsive design and user interface best practices.
