Would You Rather Game

This Would You Rather game was created using HTML, CSS, and JavaScript. The game presents users with a series of challenging questions, each requiring them to choose between two options. There are three different game modes including silly, iceabreaker, and deep. The goal of this application was to create an easy to use game to start conversations between classmates, family, friends, and more.

The game's user interface is designed with HTML and styled with CSS, providing an intuitive and visually appealing experience. The use of JavaScript allows for dynamic features such as real-time scorekeeping and social sharing, enhancing the game's social aspect.
